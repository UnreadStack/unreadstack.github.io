プライバシーポリシー

このプライバシーポリシーは、StormSmash2（以下、「当アプリ」といいます）の提供者であるStorm Smash開発チーム（以下、「当方」といいます）が、当アプリを通じて取得するお客様の個人情報および利用者情報について、その取扱いを定めるものです。

制定日：2025年11月27日

1. 取得する情報および利用目的

当アプリでは、お客様へのサービス提供および利便性向上を目的として、以下の情報を取得し、利用します。

1-1. ゲーム機能の提供のために取得する情報（ランキング機能）

取得情報:

  • **ランキング表示名（ニックネーム）:** お客様が任意で設定する文字列。
  • **ハイスコア:** ゲームプレイの記録。
  • **端末識別情報:** ユーザーを特定し、スコアを紐付けるための情報（端末情報から生成される識別子）。

取得情報の利用目的:

  • ランキング機能を提供し、他のユーザーとスコアを競うため。
  • **【重要】ニックネームとハイスコアは、アプリ内のランキング画面にて他のユーザーに対して公開されます。**
  • 不正行為の監視および防止のため。

【ニックネームに関するご注意】
ランキング表示名は自由入力ですが、**個人情報（メールアドレス、電話番号、実名など）の入力は推奨しません**。万が一、個人情報が含まれる表示名を設定され、その結果生じた損害等について、当方は一切の責任を負いません。不適切な表示名が確認された場合、ランキングから削除する場合があります。

1-2. 広告配信のために取得する情報（Google Mobile Ads SDK / AdMob）

取得情報:

  • 広告ID (Advertising ID)
  • デバイス情報 (OSバージョン、機種情報など)
  • IPアドレス
  • 広告の表示、クリック等の履歴

取得情報の利用目的:

  • ユーザーの興味・関心に基づいた広告（パーソナライズド広告を含む）を表示するため。
  • 広告効果の測定および分析のため。

提供元：Google LLC およびその広告配信パートナー。取得された情報は、Google社のプライバシーポリシーに基づいて管理されます。
Googleプライバシーポリシー

2. 機密性の高い端末機能の利用について

当アプリは、お客様のプライバシーに配慮し、**カメラ、マイク、位置情報サービス、連絡先、カレンダー、ヘルスケア情報**などの機密性の高い端末機能、およびそれらに関連する情報を**一切取得・利用しません**。

3. ターゲットユーザーとプライバシー保護について

当アプリは、主に13歳以上のお客様をターゲットユーザーとしています。

  • 13歳未満の児童の利用について：当アプリは、意図的に13歳未満のお客様（児童）からの個人情報を収集・利用することはありません。万が一、13歳未満の利用者がいると判断された場合、適用される法令およびGoogleのポリシーに基づき、パーソナライズされていない広告のみが表示されるなど、適切な保護措置が適用されます。
  • 広告の配信：当アプリからの広告リクエストには、13歳未満の対象者と判断しその旨のフラグを設定しており、パーソナライズド広告の配信を許可していません。

4. 第三者提供について

当アプリは、上記「1. 取得する情報および利用目的」に記載の機能・広告配信サービスを除き、法令に基づき開示が認められる場合を除いて、お客様の同意を得ることなく第三者に個人情報を提供することはありません。

5. プライバシーポリシーの変更

当方は、法令の改正、当アプリのサービス内容の変更等により、本ポリシーを任意に改定することがあります。改定後のプライバシーポリシーは、速やかに当方のウェブサイトに掲載します。
